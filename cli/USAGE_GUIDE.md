# Agent Monster CLI - 功能演示指南

## 项目概述

这是一个专为Agent Monster游戏设计的高级终端用户界面(TUI)客户端。使用Go语言和Bubble Tea框架构建，提供美观、快速、响应式的游戏体验。

## 🎯 核心特性

### 1. 彩色TUI界面
- 使用256色彩色支持
- 美观的菜单选择界面
- 实时响应的键盘输入
- 清晰的信息展示和错误提示

### 2. 快速导航
- 方向键/HJKL支持
- 快速返回上级菜单

## 📋 使用流程

### 1. 编译和运行

```bash
# 进入CLI目录
cd /root/pet/agent-monster/cli

# 方法一: 使用启动脚本
chmod +x launch.sh
./launch.sh build    # 编译
./launch.sh run      # 运行

# 方法二: 直接编译运行
go build -o agent-monster cmd/main.go
./agent-monster

# 方法三: 指定服务器
./agent-monster -server http://your-server:8080
```

### 2. 键盘操作

| 按键 | 功能 |
|------|------|
| ⬆️ / K | 菜单上移 |
| ⬇️ / J | 菜单下移 |
| Enter | 确认选择 |
| H / Esc | 返回菜单 |
| Q / Ctrl+C | 退出程序 |

## 🐛 故障排除

### 常见问题

**问: 界面显示混乱**
答: 
- 检查终端是否支持256色: `echo $TERM`
- 设置: `export TERM=xterm-256color`

**问: 键盘输入无反应**
答:
- 确保在TTY模式下运行
- 关闭终端输入法
- 重启终端应用

**问: 无法连接服务器**
答:
- 确保judge-server正在运行
- 检查服务器地址: `./agent-monster -server http://your-server:8080`
- 检查防火墙配置
