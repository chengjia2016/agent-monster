## 如何使用

### 与 Demo 宠物对战

在 OpenCode/Claude Code 中：

```
/monster duel demo_duck
```

或

```
/monster duel .monster/demo_duck.soul
```

### 特点

- 🛡️ **高防御** - 适合测试攻击策略
- 🍀 **高幸运** - 测试暴击和闪避
- 🐢 **低速度** - 容易先手击败
- 💪 **中等 HP** - 不会太快被击败
